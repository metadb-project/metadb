# **Contributing to Metadb**

Metadb is the core software of the [Library Data Platform (LDP)](https://librarydataplatform.org/), a community-governed open source project supported by the [Open Library Foundation](https://openlibraryfoundation.org/). We welcome contributions of all kinds.

## **Ways to Contribute**

Contributing to Metadb doesn't require writing code. Useful contributions include:

* Reporting bugs or requesting features via the [issue tracker](https://github.com/metadb-project/metadb/issues)  
* Testing pre-release versions and sharing findings  
* Participating in the [FOLIO Reporting SIG](https://folio-org.atlassian.net/wiki/spaces/RPT/overview), where community priorities for Metadb development are discussed  
* Improving documentation  
* Sustaining the project financially — see [librarydataplatform.org/funders](https://librarydataplatform.org/funders/)

## **Contributing Code**

Open an issue before beginning significant work, so the community can provide feedback early. Submit changes via pull request. The Technical Oversight Committee (TOC) reviews and approves contributions.

## **Becoming a Committer**

Contributors who demonstrate sustained, quality involvement may be nominated for Committer status by any existing Committer. The TOC evaluates nominees based on their contributions, understanding of the codebase, and collaborative approach. See the [LDP Governance Model](https://librarydataplatform.org/governance/) for details.
